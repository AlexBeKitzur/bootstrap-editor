Editors for every reason.

Many of the projects we work on at Fat Panda involve enabling users to edit and publish content. We love Bootstrap for rapid prototyping, and felt that this ecosystem could benefit from the addition of drop-in content editors, as easy to get running as adding a single <textarea> to your markup.

Currently the goal of this project is to provide editors for three kinds of content: WYSIWYG, Markdown, and source code.

Setup

WYSIWYG Editor

TinyMCE

Transform a <textarea> input into a TinyMCE editor with formatting tools, drag-and-drop uploading, and distraction-free mode.

<textarea data-edit-with="wysiwyg" placeholder="This is where you write and style!"><textarea>

Usage

Just select an <textarea> using jQuery and call

$('textarea').editor('wysiwyg');

Markup

You will use data attributes to configure your TinyMCE editor. When the DOM is ready, any <textarea> having data-edit-with="wysiwyg" will automatically become a TinyMCE editor.

Methods

$().val()

Use the familiar val() function to read and write from the editor.

<textarea data-edit-with="wysiwyg">foo<textarea>
<script>
  $('textarea').val(); // == <p>foo<p>
  $('textarea').val('<b>bar!</b>'); // HTML editor updated!
</script>

$().editor('exec', cmd)

This will invoke TinyMCE.execCommand(cmd) and so provides you with direct access to the TinyMCE API.

$('textarea').editor('exec', 'bold'); // current selection will toggled bold

Our proxy method accepts all of the parameters accepted by TinyMCE.execCommand():

$('textarea').editor('exec', cmd, ui, val);

Markdown Editor

PageDown

Transform a <textarea> input into a Markdown editor with preview, drag-and-drop uploading, and distraction-free mode.

<textarea data-edit-with="markdown" placeholder="This is where you write; forget style!"><textarea>

Code Editor

ACE

Transform a <textarea> input into an ACE code editor.

<textarea data-edit-with="source"><textarea>
